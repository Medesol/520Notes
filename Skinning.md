## Skinning
Definition: calculate how the mesh is moving consistently with the motion of the joints.

Maya don't have builtin physics except for clothes, hair and particles.

- 做两个版本的动作，一个没有物理，一个有物理模拟，物理模拟需要很长时间做完
- 之后训练神经网络输入简单动作输出精细动作
- 最后可以简单制造简单动作然后通过训练好的神经网络迅速输出精细动作，可以real time

### Pose Space Deformation
Can override skinning generated by computer to make it more beautiful. 手画几个pose的mesh之后，得到和skinning之间的delta，然后中间动作可以用这些delta做插值。

- example poses
- extrapolate and intepolate
- scattered data interpolation
	- use machine learning function (radial basis function) to solve 
	- shepard interpolation 
		- dist(p, pi) = ||p-pi||
		- 即使和某一个example pose很接近了，delta还是会受其他example poses影响

#### Radial basis function
smooth function so there is no kinks.
